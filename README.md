# Robot-xChallenge2022

Kod robota na zawody xChallenge 2022 w Rzeszowie.

## Requirements:

- [ROS2](https://docs.ros.org/en/foxy/Installation.html)
- GPIO library

```
pip install RPi.GPIO
```

## Electronic parts

- Raspberry PI
- BLCD motors 4x
- DC motors 2x
